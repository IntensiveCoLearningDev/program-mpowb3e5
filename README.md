# 财富自由理论基础共学

## 共学简介

你是否对“财富自由”充满向往，却不知从何开始？市面上理财信息纷繁复杂，容易让人迷失。本共学从零出发，为你梳理财富自由的底层逻辑，帮助你建立系统的理论知识框架。适合任何对个人财务管理感兴趣、想为后续投资实战打基础的初学者。通过4周结构化学习，你将掌握资产与负债、被动收入、复利效应、资产配置等核心概念，每周还有线上分享交流巩固所学。结营时，你能清晰表达自己的财富自由理念，并制定出初步的个人财务计划。让这次共学成为你迈向财务自主的第一步。

## 共学内容 / 课程信息

### 学习目标

掌握财富自由的基本理论知识，理解资产、负债、被动收入、复利等核心概念，能结合自身情况制定初步的财务自由计划。

### 每周安排

#### 第1周：财富自由基础概念

* **主题**：什么是财富自由？重新定义财富与工作

* **阅读材料**：

  * 《富爸爸穷爸爸》第1-5章（可购买纸质书或电子书，或参考豆瓣书评：<https://book.douban.com/subject/1033773/）>

  * 余额宝、指数基金入门介绍（参见“简七理财”公众号相关文章或“且慢”平台科普）

* **练习任务**：

  * 写一份500字左右的个人财务现状分析（收入、支出、资产、负债）

  * 列出你理解的“资产”和“负债”各3个例子

* **交付物**：个人财务现状分析文档

* **线上分享**：周日晚上20:00腾讯会议，每人5分钟分享本周学到的核心概念及困惑

#### 第2周：资产与负债，现金流象限

* **主题**：理解资产和负债的真正区别，现金流象限（ESBI）模型

* **阅读材料**：

  * 《富爸爸穷爸爸》第6-10章

  * 视频：《罗伯特·清崎讲解现金流象限》<https://www.bilibili.com/video/BV1xx411c7mQ> （若失效，可搜索“ESBI象限 解释”）

* **练习任务**：

  * 根据现金流象限，判断自己目前处于哪个象限，并规划向B/I象限的过渡步骤

  * 分析一个你熟悉的成功财富自由案例（如巴菲特、李笑来、某博主等），写成300字分析

* **交付物**：个人象限分析及案例报告

* **线上分享**：周日讨论，每人分享自己的象限分析和案例

#### 第3周：复利、被动收入与投资工具

* **主题**：复利的力量，被动收入的主要来源，常见投资工具（股票、基金、REITs等）

* **阅读材料**：

  * 博多·舍费尔《小狗钱钱》第1-8章（电子书可在微信读书搜索：<https://weread.qq.com/> ，或购买纸质书）

  * 可汗学院公开课：复利概念（英文，有中文字幕）<https://www.khanacademy.org/economics-finance-domain/core-finance/interest-tutorial/cont-comp-int/v/introduction-to-compound-interest>

  * 公众号“银行螺丝钉”关于指数基金的科普文章（搜索“指数基金投资指南”系列）

* **练习任务**：

  * 用Excel计算：假设每月定投1000元，年化收益率8%，30年后有多少钱？理解复利计算器

  * 列出3种可能的被动收入来源，并评估自己实现的可能性

* **交付物**：复利计算表格与被动收入评估报告

* **线上分享**：周日分享各自计算结果和被动收入想法

#### 第4周：资产配置与个人财务自由计划

* **主题**：如何根据自己的风险偏好做资产配置，制定个人财务自由计划

* **阅读材料**：

  * 伯顿·马尔基尔《漫步华尔街》第1-4章（关于随机漫步理论的基础，电子书可在微信读书搜索）

  * 先锋集团（Vanguard）资产配置白皮书中文版：<https://www.vanguard.com.hk/documents/asset-allocation-investing-traditional.pdf> （若链接失效，搜索“Vanguard 资产配置 白皮书”）

  * 文章：Dean Karlan《聪明的投资者》导读（哈佛大学公开课讲义链接：<https://www.edx.org/course/saving-money-smart-decisions> （英文，可看摘要）

* **练习任务**：

  * 根据现有资产和收入，制定一份简单的财务自由计划（包括目标金额、每月储蓄率、投资组合建议）

  * 写一篇300-500字的学习感悟，总结4周的收获和下一步行动

* **交付物**：个人财务自由计划书（Word或PPT）+ 学习感悟

* **结营分享**：周日腾讯会议，每人5分钟展示个人财务自由计划并分享整体感悟
